# Self-printing homepage

A live demo at https://mutoo.im

## Install

Run `yarn` to install dependencies:
```
$ yarn
```

## Dev

Run `yarn dev` to run webpack-dev-server on http://localhost:3132
```
$ yarn dev
```

## Build

Run `yarn build` to build production files in `/dist/` folder
```
$ yarn build
```
